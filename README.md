# fintrack-dash
A clean, interactive personal finance dashboard built as a single HTML file using Tailwind CSS and vanilla JavaScript. 

Features include:
• Dashboard overview with summary cards (Balance, Income, Expenses)
• Interactive charts: Balance trend (line) and Spending breakdown (doughnut)
• Full transaction list with live search and type filtering
• Role-based UI (Admin can add/delete entries, Viewer is read-only)
• Insights section with monthly comparison and observations
• Fully responsive design with smooth interactions

Made as part of a frontend development assignment to demonstrate clean UI, state management, and user experience. Open the index.html file to explore.
